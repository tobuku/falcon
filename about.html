About

Falconry Directory USA lists falconry related services and products. United States first. Global later.

How listings work

  • Free listings show basic info.
  • Verified listings show a badge after proof checks.
  • Featured listings receive top placement in select views.
  • Report flows support cleanup of bad data.

Verification rules

  • Phone and website checks.
  • Permit claims stored as text, state specific.
  • Credentials fields store level and years.
  • Public proof links stored when provided.

Global plan

  • Add country selector and country pages.
  • Add language fields per listing.
  • Add region level filters.
  • Add international permit notes per jurisdiction.
